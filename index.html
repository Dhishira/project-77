Project on Console


Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and
is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.


A console is an operating system window through which a user can communicate with the operating system
or we can say a console is an application in which we can give text as an input from the keyboard and
get the text as an output from the computer end. ... You are not allowed to inherit Console class.
